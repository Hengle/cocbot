# COC Bot
Use AutomatorX engine.

Create army automatic.

## Installation
1. Install [AutomatorX](https://github.com/codeskyblue/AutomatorX)

## Usage
1. Connect iPhone6P to a Mac.
2. Install atx-helper app, which you have to contact author.
3. Run the script

	```
	python main.py
	```
